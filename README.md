# SkillScreen

A modern full-stack interview management and live video interview platform built with Next.js, Stream, Convex, and Clerk.

SkillScreen enables teams to schedule, manage, and conduct technical interviews with real-time video calls, screen sharing, and structured candidate evaluation.


## Features

- Real-time Video Calls (Stream Video SDK)
- Screen Sharing
- Screen Recording
- Interview Scheduling and Management
- Candidate Evaluation (Pass / Fail System)
- Interview Comments and Feedback
- Authentication and Authorization (Clerk)
- Server Actions
- Client and Server Component Architecture
- Dynamic and Static Routing (Next.js App Router)
- Modern UI with Tailwind CSS and Shadcn UI


## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Stream Video SDK
- Convex (Realtime Backend and Database)
- Clerk (Authentication and Authorization)
- Tailwind CSS
- Shadcn UI
